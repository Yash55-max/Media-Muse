# 🎵 Media Muse

**AI-Powered Music Recommendation Web Application**

## 📌 Overview

Media Muse is an **AI-driven music recommendation web application** that delivers personalized music suggestions based on user preferences and contextual data. The platform integrates modern AI APIs to enhance discovery, improve engagement, and provide a smarter listening experience through a clean, web-based interface.

---

## 🚀 Key Features

* 🎧 **AI-Based Music Recommendations**
  Personalized song suggestions generated using intelligent models and APIs.

* 🌍 **Location-Aware Suggestions**
  Enhances recommendations by adapting to the user’s geographic context.

* 📝 **AI-Generated Captions & Descriptions**
  Uses generative AI to create music-related captions and contextual insights.

* 🔗 **Spotify API Integration**
  Fetches real-time music data, metadata, and recommendations.

* 📱 **Progressive Web App (PWA) Support**
  Enables fast loading, offline capabilities, and app-like experience.

* 🎨 **Responsive UI**
  Works seamlessly across desktop and mobile devices.

---

## 🧠 How It Works

1. User interacts with the web interface to explore music.
2. The system fetches music data using the Spotify API.
3. AI models analyze preferences and context.
4. Generative AI creates captions and recommendations.
5. Personalized results are displayed in real time.

---

## 🛠️ Tech Stack

* **Backend / Logic:** Python
* **Frontend:** HTML, CSS, JavaScript
* **APIs:** Spotify API, Gemini API
* **AI Capabilities:** Recommendation logic, generative captions
* **PWA:** Service Workers, Web App Manifest

## ⚙️ Setup & Installation

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/Yash55-max/Media-Muse.git
cd Media-Muse
```

### 2️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

### 3️⃣ Configure API Keys

Create a `.env` file and add:

```
SPOTIFY_CLIENT_ID=your_spotify_client_id
SPOTIFY_CLIENT_SECRET=your_spotify_client_secret
GEMINI_API_KEY=your_gemini_api_key
```

### 4️⃣ Run the Application

```bash
python app.py
```
